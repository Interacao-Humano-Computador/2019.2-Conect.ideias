# Planejamento de Melhoria
<div class="line"></div>


<p align="justify">&emsp;
Para uma melhoria do projeto, analisaremos o que fora feito, assim como o que não fora realizado, afim de mitigar os erros e trazer novos artefatos ao projeto.
</p>
