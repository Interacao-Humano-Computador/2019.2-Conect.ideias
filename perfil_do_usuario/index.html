Perfil do Usuário


  Perfil de usuário é uma descrição detalhada das características dos usuários cujos objetivos devem ser apoiados pelo sistema sendo projetado.
  A partir dos dados coletados, podemos agregar os valores em grupos e faixas na qual os usuários se encaixam, e assim traçar os perfis de usuários com características semelhantes e calcular a proporção de usuários que se encaixam em cada perfil.
